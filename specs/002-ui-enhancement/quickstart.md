## 確認手順

1. `npm run build` でエラーが出ないことを確認
2. Chrome で `chrome://extensions` を開く
3. 「パッケージ化されていない拡張機能を読み込む」で `dist/` フォルダを選択
4. Slack のカスタマイズページ（`https://{workspace}.slack.com/customize/emoji`）を開く
5. 以下を検証:
   - 画像をドロップしてリスト行にサムネイルが表示されること
   - 既に存在する絵文字名でドロップし、❌ + エラーメッセージが表示されること
   - 同じファイルを2回ドロップし、2回目が ⚠️ + スキップメッセージで表示されること
   - 全処理完了後のサマリーにスキップ件数が含まれること
