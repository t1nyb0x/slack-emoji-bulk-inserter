# クイックスタート: Slack絵文字一括登録

**フィーチャー**: `001-bulk-emoji-insert`
**作成日**: 2026-02-22

## 前提条件

- Node.js がインストールされていること
- Chrome ブラウザがインストールされていること
- 絵文字追加権限のあるSlackワークスペースにログイン済みであること

## セットアップ

```bash
# リポジトリのクローン
git clone <repository-url>
cd slack-emoji-bulk-inserter

# 依存関係のインストール
npm install

# ビルド
npm run build
```

## Chrome拡張機能のインストール（開発者モード）

1. Chromeで `chrome://extensions/` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. プロジェクトの `dist/` ディレクトリを選択する
5. 拡張機能が一覧に追加されたことを確認する

## 使い方

1. Slackのカスタマイズページ（`https://{workspace}.slack.com/customize/emoji`）を開く
2. ページ上部にドロップゾーンが表示されることを確認する
3. 登録したい絵文字の画像ファイル（PNG、JPEG、GIF）をドロップゾーンにドラッグ＆ドロップする
4. 各絵文字の登録状態がステータスアイコンでリアルタイムに表示される
   - ⏳ 待機中
   - 🔄 処理中
   - ✅ 成功
   - ❌ 失敗
5. すべての絵文字が処理されるまで待つ

## 動作確認手順

1. テスト用の画像ファイル（例: `test_emoji.png`）を用意する
2. Slackのカスタマイズページを開く
3. 画像をドロップゾーンにドラッグ＆ドロップする
4. `:test_emoji:` がSlackに登録されたことを確認する
5. Slack上で `:test_emoji:` が使えることを確認する

## 絵文字名のルール

画像ファイル名が自動的に絵文字名に変換される:

| ファイル名          | 生成される絵文字名 |
| ------------------- | ------------------ |
| `hello_world.png`   | `:hello_world:`    |
| `My Emoji.png`      | `:my_emoji:`       |
| `Cool.Logo.v2.gif`  | `:cool_logo_v2:`   |
| `テスト!emoji@.jpg` | `:emoji:`          |
